TabataXtreme is a workout timer with a difference.

As well as all the usual flexibility you'd expect in terms of sets, reps and rest period etc,
it also allows you to slow down or speed up time. So, for example, a 60s dead lift might
actually take 70 (or 50!) seconds. Whether you mention this to your clients is up to you
but shorter durations build confidence and longer ones performance.

The best place to get it is to install it directly from either Google's play store
or the F-DROID store.
