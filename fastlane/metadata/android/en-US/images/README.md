F-Droid pulls store-listing graphics from this folder automatically. Drop files in using these exact names/sizes:

- `featureGraphic.png` — 1024x500, shown at the top of the app page.
- `icon.png` — 512x512, optional (F-Droid otherwise extracts the launcher icon from the built APK).
- `phoneScreenshots/1.png`, `2.png`, ... — at least one, recommended 1080x1920 (portrait) or similar phone aspect ratio. Numbered files control display order.

No `tabletScreenshots/` or `tvScreenshots/` needed unless you want to target those form factors specifically.
